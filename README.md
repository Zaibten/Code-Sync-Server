# Code Sync Server

[![Node.js](https://img.shields.io/badge/Node.js-16.x-green)]()

This is the **backend server** for the Code Sync Application. It handles API requests from the Flutter app, performs **code error detection**, user authentication, and serves data to the **admin panel**.

---

## Features

- User authentication (login/signup)
- Code submission and error detection
- Stores generated code (optional)
- Provides APIs for Flutter frontend
- Admin panel data API endpoints

---

## Tech Stack

- **Node.js**
- **Express.js**
- **MongoDB** (or any DB you are using)
- **JWT** for authentication
- **Cors** for cross-origin requests
